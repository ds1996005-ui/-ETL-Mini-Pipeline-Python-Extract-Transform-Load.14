# -ETL-Mini-Pipeline-Python-Extract-Transform-Load.14
How ETL works practically in analytics workflows.

1.Load raw dataset

2.Create folders: raw/, processed/, output
3.Clean missing values + duplicates.
4.Standardize column names and datatypes.
5.Create derived columns (margin, segment flags).
6.Split into separate outputs (customers/orders/products).
7.Load outputs into SQLite or export as CSV.
8.Validate counts before/after transformations.
